API Basejump: A Timestamp microservice

This is the result of an assignment from the FreeCodeCamp cirriculum.

User stories:

  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Note: Natural input and output date is based on server's locale.

Example usage:

https://timestamp-ms.herokuapp.com/December%2015,%202015

https://timestamp-ms.herokuapp.com/1450159200

Example output:

{ "unix": 1450159200, "natural": "December 15, 2015" }
